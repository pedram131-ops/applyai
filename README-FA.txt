راهنمای نصب سریع در Cloudflare Pages

1) وارد Cloudflare شوید.
2) Workers & Pages را باز کنید.
3) Create application یا Create Page را بزنید.
4) گزینه Upload assets / Upload your static files را انتخاب کنید.
5) همین فایل ZIP را آپلود کنید.
6) Deploy را بزنید.

این نسخه آماده شامل ۱۲۰۰ فرصت نمونه فعال است:
- ۴۰۰ آگهی شغلی
- ۴۰۰ دانشگاه و پذیرش
- ۴۰۰ بورسیه

هر آگهی داخل سایت این موارد را دارد:
- شرح کامل فارسی
- پیش‌نیاز و مدارک
- تاریخ میلادی و شمسی
- حضوری / دورکاری / هیبریدی
- وضعیت کمک ویزا
- پذیرش بین‌المللی
- وضعیت اتحادیه اروپا یا خارج از اروپا
- لینک منبع رسمی

نکته مهم برای نسخه واقعی زنده:
برای اینکه هر ۱۵ دقیقه واقعاً از منابع رسمی آپدیت شود، باید پوشه worker را جداگانه به Cloudflare Workers وصل کنید و D1 Database + Cron Trigger + OpenAI API Key فعال شود.
این ZIP برای Cloudflare Pages آماده است و بدون تنظیمات اضافه بالا می‌آید.
